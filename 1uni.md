## Description

1UNi is a SaaS-enabled market place powered by AI making it easy for students to get matched with programs and apply to top universities.


## Tech Stack

   - **frontend**: AngularJs/JavaScript
   - **backend**: Java - NodeJs - MongoDB 
   - **CI/CD**: Shell scripts
